## Getting Started

### Full stack (recommended)

```bash
docker compose -f docker-compose.pc2.yml --env-file .env.pc2 up -d
docker compose -f docker-compose.pc1.yml --env-file .env.pc1 up -d
```

### Frontend only

```bash
cd frontend
npm install
npm start
```

### Backend tests

```bash
cd backend
pytest
```
