# Janus Backend

Python/FastAPI backend for the Janus AI Architect.

## 🚀 Quick Start

1.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the application**:
    ```bash
    uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
    ```

For full documentation, please refer to the **[Root README](../README.md)** and the **[`docs/`](../docs/)** directory.
