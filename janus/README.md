# Janus Backend

Python/FastAPI backend for the Janus AI Architect.

## Setup

1.  **Prerequisites**: Python 3.11+
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run Application**:
    ```bash
    uvicorn app.main:app --host 0.0.0.0 --port 8000
    ```
