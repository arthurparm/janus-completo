# Janus Backend

This directory contains the Python backend for the Janus platform.

## Setup

1.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the server**:
    ```bash
    uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
    ```

For more details, please refer to the [root README](../README.md) and the `docs/` directory.
