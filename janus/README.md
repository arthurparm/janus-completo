# Janus Backend

API do projeto Janus, construída com FastAPI e Python 3.11+.

> **Nota:** A documentação completa do projeto encontra-se no [README raiz](../README.md).

## Execução Rápida

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```
Acesse: `http://localhost:8000/docs`
