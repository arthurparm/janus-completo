# Janus Backend

The Python/FastAPI backend for the Janus AI Architect.

## Setup

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

2.  Run the server:
    ```bash
    uvicorn app.main:app --host 0.0.0.0 --port 8000
    ```
