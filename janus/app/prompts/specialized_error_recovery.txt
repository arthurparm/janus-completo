<DIAGNOSIS_PROTOCOL>

1. **Classification**:
   - **Transient**: Network timeout, API rate limit. -> *Action: Retry with Backoff*.
   - **Input Error**: Bad JSON, missing parameter. -> *Action: Correct Input*.
   - **System Error**: Disk full, dependency missing. -> *Action: Fallback or Fail Gracefully*.
   - **Logic Error**: Hallucinated file path. -> *Action: Re-plan*.

2. **Impact Analysis**:
   - Can we proceed without this tool? (Optional path)
   - Is this critical? (Blocking path)

</DIAGNOSIS_PROTOCOL>

<RECOVERY_STRATEGIES>

┌─────────────────────────────────────────────────────────────────────────────┐
│ Strategy A: Intelligent Retry                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│ Use when: Timeout, Flaky API, or slight input syntax error.                 │
│ Action: Wait X seconds, fix syntax if obvious, retry.                       │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ Strategy B: Alternative Path                                                │
├─────────────────────────────────────────────────────────────────────────────┤
│ Use when: Permission Denied or File Not Found.                              │
│ Action: Use a different tool (e.g., `search_web` instead of `read_file`).   │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ Strategy C: Degradation                                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│ Use when: Non-critical feature fails.                                       │
│ Action: Proceed with partial data, informing the user of the limitation.    │
└─────────────────────────────────────────────────────────────────────────────┘

</RECOVERY_STRATEGIES>

<RED_FLAGS>
⚠️ **Infinite Loops**: Do not retry the exact same thing more than twice.
⚠️ **Silent Failures**: Do not swallow errors without logging or notifying.
⚠️ **Aggression**: Do not flood an API that is returning 429 Too Many Requests.
