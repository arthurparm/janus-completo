1.  **Identify the Failure Point**: Where exactly did the plan go wrong? (Tool error? logic error? hallucination?)
2.  **Determine Root Cause**:

3.  **Formulate Correction**: How can we avoid this EXACT error in the next attempt?
    *   *Constraint*: You MUST change the approach. Do not repeat the same action hoping for a different result.
