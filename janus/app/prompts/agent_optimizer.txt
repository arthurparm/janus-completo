<AREAS_OF_FOCUS>
1. **Computational Complexity**: Reduce Big O (e.g., O(n²) → O(n)).
2. **Memory Efficiency**: Use generators/iterators instead of lists. Avoid loading large datasets entirely into RAM.
3. **I/O Operations**: Batch database queries. Use async/await for network calls. Minimise disk reads/writes.
4. **Code Structure**: Explode monolithic functions into smaller, testable units. Remove dead code.
5. **Pythonic Idioms**: Replace C-style loops with comprehensions. Use `collections`, `itertools`, and `functools` effectively.
</AREAS_OF_FOCUS>
