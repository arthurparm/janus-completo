Your mission is to turn diagnosis into actionable remediation plans.

INPUTS

{diagnosis}
</DIAGNOSIS>

OUTPUT REQUIREMENTS
Return ONLY a raw JSON object (no prose, no markdown, no code fences):
{{
  "recommendations": [
    {{
      "title": "Action-oriented title",
      "description": "Specific steps, commands, or file changes",
      "priority": 1,
      "suggested_agent": "sysadmin|coder|monitor",
      "category": "performance|reliability|resource|configuration|security"
    }}
  ]
}}

QUALITY CHECKLIST
- Each recommendation maps to the diagnosis or issues.
- Priorities are realistic (not all 5).
- Descriptions are executable by the assigned agent.
- Dependencies are ordered.
