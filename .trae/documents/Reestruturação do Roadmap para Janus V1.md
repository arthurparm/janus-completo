# Plano de Evolução do Roadmap (V1 Funcional & Científica)

Vou reescrever o arquivo `Melhorias possiveis.md` para ser um roadmap definitivo para a **Janus V1**, incorporando uma seção robusta de embasamento científico conforme solicitado.

## 1. Nova Seção: 🔬 Scientific Foundation (10+ Papers)
Adicionarei uma seção dedicada com papers essenciais que fundamentam as arquiteturas do Janus, categorizados por área:

### Reasoning & Planning
1.  **LATS (Language Agent Tree Search)** (Zhou et al., 2023): Para o módulo de planejamento avançado.
2.  **Reflexion** (Shinn et al., 2023): Para o loop de auto-correção verbal.
3.  **Graph of Thoughts (GoT)** (Besta et al., 2023): Para raciocínio não-linear em grafos.
4.  **Tree of Thoughts (ToT)** (Yao et al., 2023): Base para exploração de múltiplos caminhos.
5.  **Chain of Thought (CoT)** (Wei et al., 2022): Fundamental para prompts de raciocínio.

### Memory & Learning
6.  **Generative Agents** (Park et al., 2023): Para arquitetura de memória (Recência, Importância, Relevância) e "Sonho/Consolidação".
7.  **MemGPT** (Packer et al., 2023): Para gestão de contexto infinito via paginação de memória.
8.  **Voyager** (Wang et al., 2023): Para aprendizado contínuo e biblioteca de habilidades (Skill Library).

### Retrieval (RAG)
9.  **Self-RAG** (Asai et al., 2023): Para recuperação auto-reflexiva e crítica.
10. **HyDE (Hypothetical Document Embeddings)** (Gao et al., 2022): Para melhor recuperação semântica baseada em alucinação controlada.
11. **RAPTOR** (Sarthi et al., 2024): Para indexação recursiva em árvore (Tree-structured Retrieval).

### Multi-Agent
12. **MetaGPT** (Hong et al., 2023): Para estruturação de SOPs (Standard Operating Procedures) em multi-agentes.
13. **CAMEL** (Li et al., 2023): Para arquitetura de "Role-Playing" e comunicação comunicativa.
