# Janus Frontend

This directory contains the Angular frontend application.

**Full Documentation:** Please refer to the [root README](../README.md) for full project documentation, architecture, and roadmap.

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm start
```

- Access: `http://localhost:4200`
- API Proxy: configured to `http://localhost:8000` via `proxy.conf.json`

## Testing

```bash
# Run unit tests
npm run test

# Run linter
npm run lint
```
